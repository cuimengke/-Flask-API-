# -基于flask的调用讯飞接口的人俩特征提取与人脸比对的网页-
页面使用的是html+js+css，后端用的是Flask，调用了讯飞开放平台的模型的API，还可以实现更多功能，只需要调用讯飞平台模型就行，实习小作业。

使用方法：
首先，在讯飞开放平台注册账号
